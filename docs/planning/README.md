# 规划文档索引

本目录保存学伴图谱的版本计划、路线图和单版本执行说明。

## 阅读顺序

1. [VERSION_PLAN.md](VERSION_PLAN.md)：版本总览、当前优先级、跨版本依赖和通用验收标准。
2. [ROADMAP.md](ROADMAP.md)：按 Now / Next / Then / Later / Release 展开的执行路线。
3. [versions](versions)：每个版本的独立执行说明。

## 维护规则

- 调整版本边界时，先更新 `VERSION_PLAN.md`，再同步 `ROADMAP.md` 和对应版本文件。
- 单版本文件只写该版本要交付、要验收、暂不做的内容，不写跨多个版本的大愿景。
- 当前版本收束时，必须把完成状态同步到 `README.md`、`PROJECT_LOG.md` 和必要架构文档。
- 新增能力必须能回答：它从哪里来、写到哪里、被哪个后续版本依赖。
