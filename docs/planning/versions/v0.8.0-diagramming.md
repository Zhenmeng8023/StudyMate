# v0.8.0 工程图谱版

## 版本定位

面向软件学习者增强工程图能力，让学伴图谱不只是自由知识画布，也能创建 UML、ERD、C4、流程图和接口图等专业图。

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 数据 | `diagram_templates`、`diagram_shape_libraries`、`diagram_validations`、`diagram_sources` |
| 后端 | 模板管理、Mermaid 导入、SQL DDL 解析、OpenAPI 解析、图谱合法性校验 |
| 前台 | 模板中心、图形库面板、专业模式切换、Mermaid/SQL/OpenAPI 导入面板、校验结果面板 |
| 图谱核心 | 自由模式、UML 模式、ERD 模式、C4 模式、流程图模式 |
| 文档 | 工程图模式说明、导入格式说明、模板维护说明 |

## API 范围

- `GET /api/v1/diagram/templates`
- `POST /api/v1/graphs/:id/import/mermaid`
- `POST /api/v1/graphs/:id/import/sql`
- `POST /api/v1/graphs/:id/import/openapi`
- `POST /api/v1/graphs/:id/validate`

## 验收标准

- 用户可以从模板创建 UML、ERD、C4 或流程图。
- 用户可以导入 Mermaid 生成图谱草稿。
- 用户可以导入 SQL DDL 生成 ERD 草稿。
- 用户可以导入 OpenAPI 生成接口分组图。
- 图谱校验能返回可读问题列表。
- 专业图模式不会破坏自由知识图谱能力。
