# v0.7.0 复习 AI 版

## 版本定位

完成“理解之后记住”的闭环，让图谱、笔记、PDF 批注和 AI 输出都能进入卡片复习。AI 只产出草稿，必须经用户确认后写入。

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 数据 | `decks`、`cards`、`card_reviews`、`review_schedules`、`ai_tasks`、`ai_usage_logs`、`ai_quota_logs`、`ai_conversations`、`ai_drafts` |
| 后端 | 卡组、卡片、复习队列、复习记录、SM-2 调度、AI 任务、AI 配额、AI 草稿 |
| 前台 | 卡组列表、今日复习页、卡片编辑页、来源跳转、AI 侧边栏、AI 结果确认面板 |
| AI 能力 | 总结、解释概念、生成卡片、生成图谱分支草稿、生成学习路线草稿 |
| 文档 | AI 安全边界、草稿确认流程、复习算法说明、卡片接口摘要 |

## API 范围

- `POST /api/v1/graphs/:id/ai/expand`
- `POST /api/v1/graphs/:id/ai/summarize`
- `POST /api/v1/graphs/:id/ai/generate-cards`

## 验收标准

- 用户可以创建卡组和卡片。
- 今日复习队列根据 SM-2 生成。
- 用户完成复习后，下次复习时间正确更新。
- 用户可以从图谱节点、笔记或 PDF 批注生成卡片草稿。
- AI 生成内容必须经用户确认后写入。
- AI 任务、用量和错误状态可追踪。

## 交给下一版本

- 图谱、笔记、PDF 与卡片之间的来源引用。
- 可复用的 AI 草稿确认流。
- AI 任务记录和配额模型。
