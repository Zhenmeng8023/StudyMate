# v0.1.0 工程基础版

## 版本定位

建立学伴图谱的稳定工程底座。这个版本不追求业务完整，只保证后续模块能在清晰的目录、配置、启动和验证体系上开发。

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 数据与配置 | `.env.example`、本地上传目录占位、基础配置加载 |
| 后端 | Go + Gin 服务、`/health`、`/api/v1/health`、模块目录结构 |
| 前台 | React + Vite 用户端应用壳、基础路由和页面占位 |
| 后台 | Vue + Vite 管理端应用壳、基础路由和页面占位 |
| 共享包 | `graph-core`、`editor-core`、`api-client`、`ui` 占位 |
| 文档 | README、开发说明、架构说明、路线和版本计划 |

## 验收标准

- `go test ./...` 通过。
- `npm run typecheck` 通过。
- `npm run build:user` 通过。
- `npm run build:admin` 通过。
- 前台、后台、后端可在本地启动并访问。

## 交给下一版本

- 稳定的 monorepo 结构。
- 可复现的本地启动方式。
- 后续认证、文件、数据库模块的目录边界。

## 暂不做

- 用户登录注册。
- 数据库连接。
- 业务 CRUD。
- 图谱画布交互。
- AI 调用。
