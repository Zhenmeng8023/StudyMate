# v0.2.0 平台基础版

## 版本定位

完成所有业务模块共用的平台能力，让用户、权限、文件和基础数据访问先稳定下来。后续社区、资料、笔记、图谱和后台治理都依赖这一层。

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 数据 | `users`、`roles`、`user_roles`、`refresh_tokens`、`files`、`audit_logs` |
| 后端 | `auth`、`user`、`file`、`admin` 模块，JWT 鉴权、错误处理、请求日志 |
| 前台 | 登录、注册、个人资料页、基础路由守卫 |
| 后台 | 管理员登录页、后台鉴权入口 |
| 基础设施 | MySQL、MongoDB、Redis 连接检查，本地文件上传 |
| 文档 | 数据库连接说明、迁移说明、认证和文件接口摘要 |

## 验收标准

- 用户可注册、登录、刷新会话和退出。
- 管理员可登录后台。
- 鉴权接口未登录访问返回正确错误。
- 可上传文件并查询文件元数据。
- 数据库连接异常有清晰错误提示。

## 交给下一版本

- 可复用的鉴权中间件。
- 用户和管理员身份边界。
- 文件索引和文件访问控制基础。
- 审计日志基础能力。

## 暂不做

- OAuth 第三方登录。
- 复杂 RBAC 权限矩阵。
- 对象存储。
- 文件在线预览。
- 多租户。
