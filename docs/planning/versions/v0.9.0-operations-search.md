# v0.9.0 搜索运营版

## 版本定位

补齐产品运营、搜索、后台治理和课程雏形，准备进入正式发布前的完整试用阶段。

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 数据 | `search_sync_jobs`、`system_configs`、`courses`、`chapters`、`course_members`、`assignments` 占位 |
| 后端 | 搜索索引抽象、Meilisearch 实现、搜索同步任务、后台仪表盘、用户封禁、AI 配额调整、课程初版 |
| 前台 | 全站搜索页、课程空间初版页面、核心结果跳转 |
| 后台 | 仪表盘、用户管理、内容审核、举报处理、标签分类、AI 用量监控、系统配置 |
| 文档 | 搜索同步说明、后台运营手册、课程空间说明、Meilisearch 部署说明 |

## 搜索范围

- 帖子。
- 资料。
- 笔记。
- 图谱。
- 卡片。
- 用户。

## 验收标准

- 用户可以跨模块搜索公开或有权限访问的内容。
- 内容更新后能异步同步到搜索索引。
- 管理员可以查看核心运营数据。
- 管理员可以处理举报和审核任务。
- 管理员可以查看 AI 任务和用量。
- 课程空间可以创建课程、章节和成员关系。

## 交给下一版本

- 可演示的全站搜索能力。
- 可支撑小规模试用的后台治理能力。
- 基础课程空间数据。
- 发布前可观测的运营和 AI 使用数据。

## 暂不做

- Elasticsearch。
- 向量数据库。
- 完整 LMS。
- 个性化推荐算法。
- 多租户。
- 复杂 BI 分析。
