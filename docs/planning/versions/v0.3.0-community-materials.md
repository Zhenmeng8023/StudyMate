# v0.3.0 社区资料版

## 版本定位

让学伴不再只是空壳，先具备学习社区和资料库的可用产品闭环。此版本提供后续阅读、笔记、图谱和搜索的内容来源。

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 数据 | `posts`、`comments`、`likes`、`favorites`、`reports`、`materials`、`material_categories`、`material_tags`、`material_files`、`moderation_tasks` |
| 后端 | 社区帖子、评论、点赞、收藏、举报；资料 CRUD、分类、标签、评分、收藏；后台审核 |
| 前台 | 社区信息流、帖子详情、发帖页、资料列表、资料详情、资料上传入口 |
| 后台 | 审核列表、审核通过、驳回、下架 |
| 文档 | 社区接口摘要、资料接口摘要、审核流程说明 |

## 验收标准

- 用户可以发布帖子，并被其他用户评论、点赞、收藏。
- 用户可以创建资料并上传关联文件。
- 管理员可以审核帖子、评论和资料。
- 被下架内容前台不可见。
- 资料详情能成为后续阅读和笔记入口。

## 交给下一版本

- 可阅读的资料来源。
- 可引用的帖子和资料实体。
- 资料文件与文件索引关系。
- 后台审核基础链路。
