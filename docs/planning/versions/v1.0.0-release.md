# v1.0.0 正式发布版

## 版本定位

达到可公开演示和小规模试用的产品质量。重点从“功能堆叠”转向稳定性、体验一致性、文档完整性和闭环可用性。

## 发布范围

| 范围 | 发布要求 |
| --- | --- |
| 学习闭环 | 资料阅读、笔记沉淀、图谱组织、AI 辅助、卡片复习、社区分享可串起来 |
| 后台治理 | 用户管理、内容审核、资料管理、图谱模板、AI 任务、系统配置可用 |
| 数据与部署 | MySQL、MongoDB、Redis、Meilisearch 部署说明和迁移脚本完整 |
| 质量 | 核心路径回归测试、基础性能检查、安全检查和文件访问检查完成 |
| 文档 | 本地开发、部署、数据库迁移、API 概览、演示脚本和演示数据完整 |

## 交付切片

| 切片 | 交付内容 |
| --- | --- |
| 后端 | 核心模块接口稳定、错误码规范、统一响应结构、数据迁移脚本完整、基础安全策略 |
| 前台 | 工作台、社区、资料库、PDF 阅读器、笔记、图谱、卡片复习、AI 助手体验统一 |
| 后台 | 用户管理、内容审核、资料管理、图谱模板、AI 任务、系统配置 |
| 数据 | 演示数据、备份恢复建议、生产环境 `.env` 模板 |
| 文档 | 本地开发、部署、迁移、API 概览、演示脚本、排错说明 |

## 验收标准

- 新用户可以从注册开始完成完整学习闭环。
- 管理员可以治理公开内容和查看 AI 使用情况。
- 核心路径无阻塞级错误。
- 构建、测试、启动文档可复现。
- 至少准备一套完整演示数据。
- 核心页面具备空状态、加载态、错误态。
- 移动端具备基础适配，不出现阻塞级布局问题。

## 发布前检查

- `go test ./...` 通过。
- 前台和后台构建通过。
- 数据库迁移可从空库执行。
- 核心接口鉴权、权限和文件访问校验通过。
- AI 用量、失败和超限状态可见。
- 演示脚本可以按步骤复现。

## 暂不做

- 大规模商业化部署。
- 多人实时协作。
- 桌面端 Tauri。
- 移动原生 App。
- 插件市场。
- 完整 RAG 向量知识库。
